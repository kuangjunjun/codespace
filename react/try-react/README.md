# react 了解

- vue 和 react 有什么相同点和不同点
  - 都是MVVM架构的现代前端框架，React早于Vue
  - VDOM diff 组件化

  - VUE上手简单，主要得益于良好的API
  - REACT JSWAY
    - 组件就是函数，返回html就是组件
    - JSX
  - vue 支持指令 v-model 双向数据绑定
    react觉得双向绑定性能特别不好，不支持

- 为什么要用ts
  - JS是弱类型语言， ts 添加了类型系统
  - 减少很多错误出现
  - 不需要的代码
      需求改变后，有些代码没有删除
  - 类型推导
  - 静态编译
